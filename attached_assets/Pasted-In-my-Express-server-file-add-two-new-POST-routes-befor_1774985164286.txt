In my Express server file, add two new POST routes before the app.listen() call.

Route 1: POST /api/speak
- Reads ELEVENLABS_API_KEY and ELEVENLABS_VOICE_ID from process.env
- Accepts { text } in the request body
- Strips HTML tags from text and limits it to 2000 characters
- Makes a POST request to https://api.elevenlabs.io/v1/text-to-speech/{VOICE_ID}
- Sends xi-api-key header with the API key
- Request body: { text, model_id: "eleven_monolingual_v1", voice_settings: { stability: 0.55, similarity_boost: 0.85, style: 0.2, use_speaker_boost: true } }
- Streams the audio/mpeg response back to the client
- Returns 500 with { error } on failure

Route 2: POST /api/ridge/chat
- Reads ANTHROPIC_API_KEY from process.env
- Accepts { messages } array in the request body
- Makes a POST request to https://api.anthropic.com/v1/messages
- Uses headers: x-api-key, anthropic-version: "2023-06-01", Content-Type: application/json
- Body: { model: "claude-sonnet-4-20250514", max_tokens: 1000, system: [the system prompt below], messages: last 14 messages }
- Returns { reply } with the text response
